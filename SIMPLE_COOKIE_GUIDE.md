# 簡單的 Cookie 提取指南

## 最簡單的方法 - 只需複製貼上

### 在 Option Samurai 頁面執行以下步驟:

1. **按 F12** 打開開發者工具
2. **點擊 Console 標籤**
3. **貼上並執行這行代碼:**

```javascript
document.cookie
```

4. **複製輸出結果** (會是一長串文字)
5. **貼給我**

就這麼簡單!我會自動處理剩下的部分。

---

## 示例

執行後你會看到類似這樣的輸出:
```
"cookie1=value1; cookie2=value2; auth0=very_long_token_here; ..."
```

把整個輸出複製貼給我即可!
